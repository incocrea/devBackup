Customize Style

This template is built with development tools: Yarn, Gulp, Twig

Make sure all development tools is installed, then you are ready to install all dependencies necessary for Mimity development.

yarn install

After installaltion is finished, feel free to edit the file related to the component you want to customize. All source files are stored in the folder 'src'

gulp watch

The above command will watch all files changed from 'src' folder and build it to 'dist' and 'html' folder

* If you want to check if there are new version of dependencies, then run the command yarn outdated, then update the latest version by adjusting the version number to the package.json file

* After that do not forget to run the gulp plugins command, this will copying required dependencies from 'node_modules' to 'plugins' folder

* If you want to rebuld all files, run gulp command without any parameter.
